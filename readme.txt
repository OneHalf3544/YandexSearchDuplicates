Программа поиска одинаковых вакансий.
Написана как решение задачи для собеседования в yandex.

Программа собирает информацию с нескольких сайтов, затем ищет
наиболее похожие. 

Для определения "похожести" учитывается:
* Название компании, разместившей вакансию;
* Регион, в котором предлагается вакансия;
* Название вакансии;
* Описание вакансии.

Для сбора информации с сайтов используется библиотека WebHarvest.
С помощью этой библиотеки данные собираются в xml-файл.

Далее с помощью SAX программа считывает собранные данные в память
и ищет схожие вакансии. 

После этого в памяти создается DOM-структура
нового xml-документа, которая записывается в файл с помощью 
XSL-преобразования. 

Записанный файл представляет из себя html-страницу 
с таблицей, в которой собраны повторяющиеся данные и ссылки на 
оригинальные веб-страницы с данными вакансиями.
